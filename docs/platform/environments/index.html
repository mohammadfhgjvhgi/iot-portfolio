خطوات التثبيت والإعداد

1
تحميل البرنامج

قم بزيارة arduino.cc/en/software وحمّل النسخة المناسبة لنظامك (Windows/Mac/Linux). النسخة الحالية هي Arduino IDE 2.x.

2
تثبيت البرنامج

شغّل ملف التثبيت واتبع الخطوات. في Windows اضغط Next ثم Install. في Mac اسحب الأيقونة لمجلد Applications.

3
إضافة دعم ESP32

افتح File > Preferences، أضف الرابط https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json في خانة Additional Boards Manager URLs.

4
تثبيت لوحة ESP32

افتح Tools > Board > Boards Manager، ابحث عن esp32 واضغط Install على ESP32 by Espressif Systems.

5
تثبيت المكتبات

افتح Tools > Manage Libraries، ابحث عن المكتبات المطلوبة مثل DHT sensor library و Adafruit SSD1306 واضغط Install.

6
اختبار أول مشروع

اختر اللوحة من Tools > Board، اختر المنفذ من Tools > Port، اكتب كود Blink واضغط Upload.
